Изображение слева

Чтобы получить изображение с левого края необходимо добавить класс left к тегу img, т.е. <img class="left" src="..." />.

Изображение справа

Чтобы получить изображение с правого края необходимо добавить класс right к тегу img, т.е. <img class="right" src="..." />.

Чтобы сбросить обтекание картинок, следующий за ними текст должен быть с классом clear, например <p class="clear">

Изображение слева с помощью тега <img>

Чтобы получить изображение с левого края необходимо добавить атрибут align="left" к тегу img, т.е. <img align="left" src="..." />.

Изображение слева с помощью тега <img>

Чтобы получить изображение с правого края необходимо добавить атрибут align="right" к тегу img, т.е. <img align="right" src="..." />.
